## 目标与范围
- 目标：收集 Kevin Warsh 的公开言论与讨论资料，构建政策思路的宏观研究体系，并推演投资框架/体系。
- 资料范围：演讲稿、证词、论文、采访/播客文字稿、视频与媒体评论。
- 证据优先级：官方原文/官方档案 > 权威机构访谈 > 媒体评论/二手解读。

## 数据采集与治理
- 来源清单（优先级）：
  - FRASER（历史演讲/声明 PDF 与元数据）
  - Hoover Institution 公开资料与活动页面
  - Fed History/官方档案与履历页
  - 主流媒体采访与播客（含转写）
- 统一元数据字段：
  - 标题、日期、来源、类型（演讲/采访/评论）、原文链接、作者/主持、摘要、主题标签、可信度等级、是否一手来源

## 研究框架（政策思路抽象）
- 主题维度：
  - 货币政策立场（鹰/鸽）
  - 通胀与增长权衡
  - 金融稳定与监管态度
  - 央行独立性与财政边界
  - 全球与汇率/资本流动
  - 市场功能与流动性
- 观点结构化：观点 → 依据 → 条件 → 结论
- 反应函数：
  - 触发条件（通胀、增长、金融压力、财政风险）
  - 工具偏好（利率/资产负债表/沟通）
  - 容忍区间与优先级排序

## 投资框架映射
- 宏观因子：利率路径、期限利差、信用利差、美元、风险偏好
- 资产响应：股/债/商品/美元/行业风格的方向性与择时
- 规则化输出：
  - 情景触发 → 组合倾向 → 风险阈值 → 对冲/退出规则

## 仓库结构建议
- sources/ 原始来源与元数据
- transcripts/ 转写与整理文本
- notes/ 研究笔记（按主题或年份）
- frameworks/ 政策框架与投资框架
- outputs/ 时间线、结论摘要、情景表
