# 月度再平衡清单

**用途**: 月度组合再平衡的操作指南
**时间**: 30–60 分钟
**频率**: 每月约 15 日（CPI公布后）

---

## 快速开始

**3步流程**:
1. 数据收集（15分钟）
2. 场景识别（10分钟）
3. 组合调仓（30分钟）

## 第二步：场景识别（10分钟）

### A. 判断通胀路径（主信号）

```
CPI <2%         → B1/B2（低通胀）
CPI 2-4%        → 过渡区
CPI 4-5%        → A1/A2（预警区）
CPI >5%         → A1/A2（高通胀）
CPI >9%         → 危机区（最坏）
```

### B. 判断增长路径（次信号）

```
GDP >2% 且 失业率 <5%   → 稳增长
GDP <2% 且 失业率 >5%   → 增长走弱
GDP <0% 或 失业率 >6%   → 衰退风险
```

### C. 检查金融稳定性

```
若 2+ 个Panic指标触发 → C1（流动性冲击）
若信用利差大幅走阔  → C2（信用压力）
```

### D. 应用边缘规则

- ☐ 应用 10% 缓冲区
- ☐ 大幅场景切换需 2 个月确认
- ☐ 应用优先级（C1 > C2 > A2 > A1 > B2 > B1）
