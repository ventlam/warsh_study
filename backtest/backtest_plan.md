# Warsh框架历史回测计划

**创建**: 2026-02-02
**周期**: 2021-01 至 2024-12（4年）
**目标**: 用真实市场数据验证框架有效性

---

## 目标

量化验证：Warsh投资框架在2021–2024期间能否相对60/40实现+5%~7%的年化超额收益，重点检验2021–2022通胀周期。

---

## 区间选择理由

**选择2021–2024的原因**:
- ✅ 覆盖关键通胀周期（Warsh预警、Fed延迟）
- ✅ 多次场景切换（A1 → A2 → B1）
- ✅ 真实压力测试（CPI峰值9.1%、2022年SPY -18%）
- ✅ 数据足够新，接近当前制度环境

## 数据需求

### 宏观信号（按月）
- **CPI**: FRED CPIAUCSL（同比）
- **核心CPI**: FRED CPILFESL
- **GDP**: FRED GDP（季度数据，月度插值）
- **失业率**: FRED UNRATE
- **Fed资产负债表**: FRED WALCL（周频 → 月均）

### 资产价格（日频 → 月频）
- **SPY**: 标普500 ETF（权益基准）
- **TLT**: 20Y+ 国债ETF（久期）
- **IEF**: 7–10Y 国债ETF（中久期）
- **TBT**: 2倍做空20Y（做空久期工具）
- **DXY**: 美元指数
- **HYG**: 高收益债ETF
- **GLD**: 黄金ETF
- **现金**: 0%（简化假设，可用DFF/12替代）

## 方法论

### Phase 1：场景识别（按月）

每个月（2021-01 到 2024-12）：

**Step 1: 计算指标**
```
CPI YoY = (CPI_t / CPI_t-12) - 1
失业率 = UNRATE
GDP增速（4Q滚动）= 估算
Panic指标 = 6项检查
```

**Step 2: 场景映射**
```
CPI >5% 且 失业率 <5%   → A1（高通胀+稳增长）
CPI >5% 且 失业率 >5.5% → A2（高通胀+弱增长）
CPI 2-4% 且 失业率 <5%  → B1（软着陆）
CPI <3% 且 失业率 >5.5% → B2（通缩性衰退）
2+ Panic指标触发         → C1（流动性冲击）
信用利差 >300bp         → C2（信用冲击）
```

**Step 3: Warsh声明标记**
- 2021-06: WSJ预警
- 2021-12: “Fed是元凶”
- 这些月份标记为“Warsh dissent”

### Phase 4：绩效指标

计算月度收益后，输出：

1. **年化收益**（CAGR）
2. **波动率**（年化）
3. **Sharpe**（假设Rf=0）
4. **最大回撤**
5. **Calmar**（收益/回撤）
6. **Alpha**（Warsh - 60/40）
7. **胜率**（战胜60/40的月份）

**关键年份分析**:
- 2022年收益（验证年）

## 预期结果

### 假设
- **2021**：Warsh提前抬高现金
  - 上半年可能跑输
  - 下半年预计跑赢

- **2022**：短久期 + 低权益
  - 显著跑赢
  - 目标：Warsh跌幅 0–5%

- **2023–2024**：逐步恢复风险
  - 可能跑输牛市
  - 但波动率更低

**总体预期（2021–2024）**:
- Warsh CAGR: 5–8%
- 60/40 CAGR: 2–4%
- Alpha: +3–4%
- Sharpe: Warsh 0.7–0.9

## 风险因素

1. **前瞻偏差**: 使用公布前CPI
   - 对策：按公布日期对齐

2. **过拟合**: 用历史数据优化配置
   - 对策：配置必须来自框架规则，不做拟合

3. **执行可行性**: TLT做空规模限制
   - 对策：在限制中记录

4. **交易成本**: 频繁再平衡
   - 对策：统一计入0.1%估算
